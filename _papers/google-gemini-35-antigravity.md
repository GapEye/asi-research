Google's Gemini API changelog is unusually useful here because it records the
developer-facing surface. On May 19, 2026, Google released `gemini-3.5-flash`,
launched Managed Agents in public preview, and released the general-purpose
Antigravity Agent preview.

## Why it matters

The Antigravity Agent can plan, reason, write and execute code, manage files,
and browse the web inside a sandbox container. That makes it a direct entry in
the agentic coding race rather than a generic chat-model update.

## ASI relevance

ASI-relevant progress increasingly appears as a bundle: stronger model, hosted
agent runtime, sandbox, browser, tools, and developer API. The model alone is
not the product; the compounding loop is.
